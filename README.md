# INLY - Pages de vente pour vendeurs informels

Système de pages web simples pour vendeurs WhatsApp en Côte d'Ivoire.

## Structure

- `templates/` : Modèles de pages disponibles
- `vendors/` : Pages des vendeurs

## Templates disponibles

1. **Dark** : Design sombre, moderne
2. **Premium** : Design haut de gamme avec hero
3. **Standard** : Design simple et efficace
4. **Premium 2** : [À décrire]
5. **Premium 3** : [À décrire]

## Comment créer une page vendeur

1. Dupliquer un dossier dans `vendors/`
2. Modifier les informations dans le JavaScript
3. Ajouter le logo
